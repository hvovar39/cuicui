*par défaut*
F1 -> fermer l’application
F2 -> sauvegarder le fichier courant
F3 -> charger un nouveau fichier
F4 -> fermer un fichier
F5 -> commencer une sélection


Ces commandes peuvent être changées dans le fichier .config mais seuls les chiffres associés aux touches -fonction- doivent être modifiés.

Pour le bien commun, évitez d’utiliser des caractères spéciaux dans ce programme.

COMMANDES :
*) changer de buffer :
	- sous LINUX : CTRL+KEY_LEFT // CTRL+KEY_RIGHT
	- sous macOS : CTRL+SHIFT+KEY_LEFT // CTRL+SHIFT+KEY_RIGHT
Les fonctions ne fonctionnent que sous mac OS :
*) copier : CTRL+B
*) couper : CTRL+N
*) coller : CMD+V
